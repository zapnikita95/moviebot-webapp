# Размещение на Cloudflare Pages

## Быстрая настройка:

1. Зайдите на https://pages.cloudflare.com/
2. Войдите через GitHub
3. Выберите репозиторий `moviebot-webapp`
4. Настройки сборки:
   - **Framework preset:** None
   - **Build command:** (оставьте пустым)
   - **Build output directory:** (оставьте пустым или укажите `/`)
5. Нажмите "Save and Deploy"
6. После деплоя получите URL вида: `https://your-project.pages.dev`
7. Укажите этот URL в BotFather: `/setmenubutton` → URL

## Преимущества Cloudflare Pages:
- ✅ Быстрый CDN по всему миру
- ✅ Автоматический HTTPS
- ✅ Бесплатный тариф
- ✅ Автоматический деплой из Git
- ✅ Отличная производительность для Mini Apps
